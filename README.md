🏦 Banking Knowledge Representation System Using NLP

## Overview

An AI-powered Banking Knowledge Representation System developed using Python, NLP, Streamlit, and Semantic Networks. The application extracts banking entities and relationships, visualizes semantic networks, performs rule-based reasoning, and provides an interactive banking chatbot.

## Features

- Natural Language Processing (NLP)
- Entity & Relation Extraction
- Semantic Network Visualization
- Frame Representation
- Rule-Based Expert System
- Banking Knowledge Base (100+ concepts)
- AI Banking Chatbot
- Streamlit Web Interface

## Technologies

- Python
- Streamlit
- NLTK
- NetworkX
- Matplotlib
